## 使用方法
Vital向けのウェーブテーブル用音声データを生成します。
256周期の音声データが生成されるので、Vitalでキーフレーム1（0ではないことに注意）と最大の256の2つに設定し、後者のPOSITIONを最大値にしてください。BLEND STYLEはNONEです。

## 波形について
n倍音の強さががパラメータ `a (0 < a < 1)` に依存した次の式で決定されます。
$$
A(n) = 1 / (-\ln(-a + 1) + 1)^n
$$
最低倍音が1でないものは、最低倍音未満の倍音が完全にカットされます。

指数関数的な倍音の減衰は、反比例で減衰するのこぎり波などと比較してアコースティックな楽器の特性により近い音色を持ちます。
